# mengniu-get-milk
蒙牛世界杯每天抢牛奶脚本（原创脚本，仅供学习交流使用，每日更新，关注忒星不迷路）

2022-12-09 更新至1.3版本

2022-12-10 更新至1.4版本

使用方法：

1.安装python

2.安装python模块 cmd执行命令

pip install pycryptodome

pip install requests

3.修改config.json里面的token即可使用 多开 复制多份exe+配置文件 放在不同文件夹即可

4.双击打开py或者用exe即可运行

# mengniu-get-milk
蒙牛世界杯每天抢牛奶脚本（原创脚本，仅供学习交流使用，每日更新，关注忒星不迷路）

2022-12-09 更新至1.3版本

使用方法：
1.安装python
2.安装python模块 cmd执行命令
pip install pycryptodome
pip install requests
3.双击打开py即可运行

说明：
活动微信小程序：蒙牛世界杯

在代码py里修改token成你的即可正常运行（token自行抓包获取）

    token是小程序包的请求头的Authorization: 
    '''
